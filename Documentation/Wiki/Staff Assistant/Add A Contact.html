Open PIB Access database.

    
Search in Contacts Form in the database to see if the contact is already entered. Easiest to choose the most unusual name and search for that.

1. Click the field you want to search by. Can be done while in any contact record.

2. Click the binoculars.

3. Enter the name to search by.

4. Click "Any Part of Field" under Match.

5. Hit your "Enter" key or click "Next" to go through the database until you find the person you want.

  To see if the registrant is already in the database to avoid duplicates.
 If not in database, add them.

 1. Click "Next" button (right-facing arrow) to go to last record added.

2. Make mental note of the Contact ID number.

3. Click the + sign to add a new record.

4. In the ContactID box type in the ID number using the previously remembered ID # and add 1.

5. Enter the remaining information into the form.

6. If you do not have a mailing address, change the Mailing List Box to N (or no).

7. See next step below to add the Seminars they want to receive emails for.

  
Click the Back arrow, then the Next arrow buttons to get the new contact to be saved.

1. After doing the Back & Forward arrows, click on one of the Seminar buttons to enable all the check boxes.

2. If they have specified a seminar to receive information on, click on the boxes they don't want to receive emails for to remove the checkmark.

 There isn't an Auto-number set up on this table due to the way the database was originally set up. A great many changes would need to be made in numerous files to do auto-number, so we are keeping it this way for the time being.
