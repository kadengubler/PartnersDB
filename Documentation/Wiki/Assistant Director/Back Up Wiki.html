Back Up Wiki
  • Go to https://partnersusu.pbworks.com/settings/backup
  • Select include current versions of all pages
  • Click Download Backup
  • Save the file to the Z drive
    • In the folder Standard Work Backup
  
 Back Up EditGrid
  • Go to http://www.editgrid.com/workspace
    • Log in
  • Click on the Standardized Work file
  • Go to File
    • Save As
    • Name the File Backup and today's date.
    • Ex. Backup 4-12-09
  • Only keep the two most recent back ups
