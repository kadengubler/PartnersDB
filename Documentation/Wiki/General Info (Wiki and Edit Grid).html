The wiki is hosted by PBWorks, and all of the moving tables with equations are hosted by EditGrid.

 

Edit Grid's Website www.editgrid.com

 

To get the spreadsheet to fill the whole browser window click on the downward facing arrow button in the top left had corner and select full screen. The spreadsheet will open in a seperate window.

 

EditGrid operates in PBWorks through a plugin, in order to get this plugin to work use the following instructions.

This plugin can be put on any page in PBWorks.

  • Edit the page that you want the plugin on.
