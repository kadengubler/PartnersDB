Changing a Page Name for a linked page

Changing a page name breaks the links, so it is not recommended, but if it needs to be done here is how.

  • In the edit function in PBWorks, click change name
  • Change the name
    • Save it
  • If it is linked in EditGrid follow the instructions below, if not skip to the linking a page important step. Delete the old link before you add the new one.
  • When you are back to the page copy the URL
  • Go to the EditGrid, and in tasks find the page that you changed
  • Change the name
  • Edit the link
    • Copy the new link in place of the old one
  • Copy that cell with the updated link
  • Paste the cell everywhere else that has the same cell
  
Uploading a New File
  • Click on the upload files button
  • Select the file you would like to upload
  
Linking to a File or Page on the Wiki
  • Enter the edit mode on the page that you would like the link to appear
    • Click the edit tab at the top of the page
  • On the right side of the page, there is an insert links box, this box has two tabs: 1) Pages and 2) Images and Files
    • Select pages if you want to link to another wiki page
    • Select Images and Files if you would like to link to an image or file
  • Insert a link
    • Go to the location on the page where you would like a link placed
    • In the insert links box click on the page or file title that you would like inserted.
  
Uploading a New Revision of a File
  • Name the file the same as it appears in the wiki
    • when you download a file from the wiki, it will add some stuff instead of spaces in the file name, so the name needs to be changed back to the original.
  • Upload it
  • To check it you can look at the page in the pages and files section to see if the revision changed
  • If you just up-rev a page, the link stays put, so this is really the way to edit files.
