Make a page for the Assistant Director

Do this in a full screen EditGrid screen (push the downward facing arrow button in the upper left hand corner of the EditGrid, and select full screen)

  • Right click on the Assist Dir Template tab and copy the sheet.
  • Rename the tab the name of the assistant director
  • DO NOT CHANGE OR DELETE THE Assist Dir Template sheet
  • Open the Task tab
  • Copy all of the tasks from column C to G and from the top of the assistant dir seminar tasks to the bottom of the seminar tasks
  • Paste the tasks into the C6 of the new sheet created
  • Paste one set of tasks for each seminar. One set after another
  • Copy any of the other assistant director tasks that the assistant director wants
  • Paste them into the assistant director's sheet.
  • If there aren't enough rows of equations, drag columns H through O down as far as needed.
  • Hide Columns D through K

 

 

 

 

 

 

 

 

 

 

  • Keeping this standard makes it easier for future administrators to follow the standard work found here. It also allows for mistakes without ruining the template.
Correct the information on the sheet On the top of the sheet there are events 1-15 one is a weekly event, and one is a monthly event, these two recurr each week or month.
  • Fill in the dates and names on the top of the sheet
    • Keep the names short
    • Don't change the days until seminar row
    • Don't change anything on the weekly or monthly events
    • Don't change the numbers 1-15 at the top
  • On Column A, input the number of the event for each seminar. If FSB were under number 1 on the events section, I would put a number 1 next to one entire set of seminar tasks. Do the same for all seminars
  • Fill in the numbers in Column A for all of the other events

 

 

 

 

 

 

  • Changing this information will ruin the equations for the recurring events

 

 

 

 

  • These numbers are what make the equations work with the dates at the top
Sort the rows
  • Select Columns A through O and Rows 6 to the end.




  • Click Data then Sort
You can sort this however you would like. A couple of common ways are shown below.
  • To sort by seminar (column A) and then Red deadline (column M)
  • To sort by deadline, sort by Red Deadline (Column M)
  • If all of the columns aren't sorted, it will mess up equations, if all of the rows aren't sorted, it will be out of order (one section will be sorted and the rest won't)
